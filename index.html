Comment prendre en main l'app

Version enfant — gros boutons, fond noir, lecture super lisible

1. Changer les logos

Touchez le logo d'une équipe pour faire défiler les 3 animaux disponibles. Les noms des équipes peuvent être modifiés directement.

2. Ajouter / corriger les points

Utilisez les grands boutons +1, +2, +3. Le bouton -1 sert à corriger une erreur.

À chaque point ajouté ou retiré, l'écran fait un flash blanc puis un clignotement rapide.

3. Régler le chrono et le buzzer

  • Entrez les minutes et secondes.
  • Appuyez sur Définir pour appliquer le temps.
  • Start lance ou met en pause le chrono.
  • À 00:00, un buzzer sonore retentit automatiquement.

4. Reset

Reset chrono remet le chrono à la valeur définie.

Reset match remet les scores à zéro et replace le chrono sur la valeur définie.

5. Autres fonctionnalités

  • Sauvegarde locale automatique des scores, noms, logos et temps.
  • Interface conçue pour être lisible sur iPad au bord du terrain.
  • Le buzzer est joué avec le moteur audio du navigateur. Si le son ne démarre pas, touchez un bouton d'abord et vérifiez que l'iPad n'est pas en silencieux.

6. Astuce iPad

Ouvrez la page dans Safari puis utilisez Partager → Ajouter à l'écran d'accueil pour l'utiliser comme une mini app.

Astuce : pour un accès en 1 clic, héberge ce fichier sur GitHub Pages puis ajoute-le à l'écran d'accueil.
