# Procédure GitHub Pages — Basket Score Kids

## 1) Créer le dépôt
- Créer un dépôt GitHub public, par exemple `basket-scoreboard-kids`
- Ajouter le fichier `basket-scoreboard-kids.html`
- Renommer le fichier en `index.html` dans le dépôt GitHub

## 2) Activer GitHub Pages
- Ouvrir `Settings > Pages`
- Source : `Deploy from a branch`
- Branch : `main`
- Folder : `/ (root)`

## 3) URL d'accès
- L'URL sera de la forme : `https://<user>.github.io/<repo>/`

## 4) Mise à jour
- Modifier `index.html`
- Commit / push
- GitHub Pages redéploie automatiquement

## 5) iPad
- Ouvrir l'URL dans Safari
- Partager > Ajouter à l'écran d'accueil
