Comment utiliser l'app — v3.1

1. Changer les logos

Clique sur un logo pour le sélectionner.

Le logo sélectionné est entouré.

2. Ajouter / retirer des points

Utilise +1, +2 ou +3 pour ajouter des points.

Utilise -1 pour enlever un point.

L'écran clignote à chaque point marqué.
